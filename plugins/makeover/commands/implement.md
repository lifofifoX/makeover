# Makeover Implement

Implement an approved theme from proposals.

## Dependencies

- **frontend-design plugin**

## Arguments

`$ARGUMENTS` = theme name to implement (e.g., "midnight-cinema")

## Workflow

1. Verify proposal exists at `tmp/makeover/themes/{name}.html`
2. Read and follow `skills/makeover/IMPLEMENT.md`
